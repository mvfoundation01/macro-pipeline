FRED API key setup (cho 3 chỉ số UNRATE/PAYEMS/FEDFUNDS)

  1. Đăng ký miễn phí tại FRED API (~ 1 phút)

1. Quy trình 4 bước

  1. Tải template Excel — ở trang chủ, mỗi vùng upload có liên kết “Tải template”. Tải 3 template (yield-curve, credit-spreads, sentiment).
  2. Điền dữ liệu vào template — mở từng file trong Excel, đọc sheet instructions, sau đó nhập dữ liệu vào sheet data. Tối thiểu 12 hàng dữ liệu (1 năm theo tháng) là đủ; nhiều hơn càng tốt.
  3. Upload và điền form — quay lại trang chủ, upload 3 file Excel (tùy chọn — có thể bỏ qua), điền 8 chỉ số macro số (PMI, CAPE, ...), chọn horizon.
  4. Bấm CHẠY FORECAST — chờ 5–30 giây. Báo cáo sẽ hiển thị tự động.

2. Nguồn dữ liệu khuyến nghị

Yield curve (2Y & 10Y Treasury)
• FRED: DGS2 (2-Year) + DGS10 (10-Year)
• TradingView: tìm US02Y và US10Y
• Khi 2Y > 10Y → yield curve đảo ngược → tín hiệu suy thoái (recession_p tăng).
Credit spreads (Investment-grade OAS)
• FRED: BAMLC0A0CM (ICE BofA US Corporate Index OAS)
• Đơn vị: basis points (bps); 100 bps = 1%
• Trên 200 bps thường báo hiệu stress; trên 400 bps là khủng hoảng.
Sentiment (AAII / NAAIM / ISM ...)
• AAII bull/bear: aaii.com/sentimentsurvey
• NAAIM: naaim.org/programs/naaim-exposure-index
• ISM Manufacturing: ismworld.org
• Trong template, mỗi hàng là một quan sát (date / indicator / value).
8 chỉ số số (form chính)
• PMI Manufacturing & Services: ISM (US) hoặc S&P Global
• CAPE Ratio (Shiller P/E): multpl.com/shiller-pe
• S&P 500 giá: TradingView / Yahoo Finance
• Nonfarm Payrolls + Unemployment: BLS Employment Situation
• Core CPI YoY: BLS CPI report (loại trừ food + energy)
• Fed Funds Rate: FRED FEDFUNDS (effective rate)

4. Hiểu Confidence vs Conviction

Confidence (0–1) phản ánh mức độ tin cậy về chất lượng dữ liệu + sample size + regime stability. Bị cap theo horizon (1Y ≤ 0.85; 3Y/5Y ≤ 0.80; 10Y ≤ 0.70 hoặc 0.55 nếu regime-stratified).

Conviction (1–10) phản ánh risk/reward asymmetry: có chỉ báo + mặt tích cực, trừ đi rủi ro + bất đối xứng xấu. Conviction có thể thấp hơn confidence khi rủi ro lớn (Vision §4).

5. Khắc phục lỗi thường gặp

  • “Thiếu cột bắt buộc” — kiểm tra dòng tiêu đề Excel; phải đúng tên cột (date, 2y, 10y, ...). Phân biệt chữ thường/hoa không quan trọng.
  • “Giá trị không hữu hạn” — có ô NaN hoặc Infinity. Xóa hàng lỗi hoặc nhập số hợp lý.
  • “File trống” — tối thiểu cần 1 hàng dữ liệu (sau header).
  • Form từ chối số — dùng dấu chấm cho thập phân (4.0 không phải 4,0).
  • Browser không tự mở — mở thủ công http://localhost:8000.

6. Data Snapshot (L11) — Vì sao có nó và làm thế nào để rebuild

Từ L11 trở đi, web app này dùng data snapshot bundled trong PyInstaller (.exe) để derive ForecastInputs từ các panel lịch sử thực (Shiller, SPX, VIX, FRED macro, NBER USREC...) thay vì dùng heuristic modulators như L10. Snapshot là một thư mục macro_pipeline/data_snapshot/ chứa ~35 panels parquet (~7-8 MB).

Trang Kết quả mới nhất có phần DATA PROVENANCE cho biết:

  • Chế độ: producer_derived (panel-based, tốt) hoặc heuristic_fallback (snapshot mất, fallback sang L10)
  • Ngày build snapshot: nếu cũ hơn 90 ngày, kết quả ít chính xác hơn
  • Producers chạy + panels sử dụng: minh bạch về data source

Rebuild snapshot khi dữ liệu lịch sử cập nhật

Nếu bạn có dev environment với data/cache/ đã refresh từ FRED + Yahoo:

  1. Mở terminal trong thư mục repo
  2. Chạy: python scripts/build_data_snapshot.py
  3. (Tùy chọn) Kiểm tra checksums: python scripts/build_data_snapshot.py --check
  4. Rebuild .exe: python scripts/build_standalone.py

Hoặc xem manifest hiện tại: python scripts/build_data_snapshot.py --list

7. FAQ

Q: Tôi có thể bỏ qua phần upload Excel không?

Có. Khi không có upload, hệ thống dùng giá trị mặc định cho yield curve / credit spreads / sentiment. Kết quả vẫn ra nhưng kém chính xác hơn so với có dữ liệu.

Q: Forecast chậy bao lâu?

Aggregator chạy khoảng 5–30 giây. Render báo cáo HTML thêm 1–3 giây nữa.

Q: Dữ liệu của tôi có an toàn không?

Có. Web app chạy hoàn toàn cục bộ trên máy bạn (localhost:8000). Không có dữ liệu nào được gửi ra ngoài.

Q: Forecast tôi nhận được có phải là khuyến nghị đầu tư không?

Không. Đây là institutional forecasting framework phục vụ research + risk management. Mọi quyết định đầu tư cần đánh giá thêm.
